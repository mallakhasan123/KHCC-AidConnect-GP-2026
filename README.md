# KHCC AidConnect — Graduation Project 2025/2026

A Smart, Human-Centered Donor and Patient Matching System for Cancer Financial Aid at KHCC using KNN and Business Intelligence.

## 📊 Results
- KNN Mean Precision@4 = **1.00** (target was 0.75) ✅
- Programme Recommender Coverage = **100%** (111/111 patients) ✅

## 📁 Files in this Repository
| File | Description |
|------|-------------|
| 01_loading.ipynb | Data loading and initial audit |
| 02_Data_Cleansing.ipynb | Null imputation, encoding, MinMaxScaler |
| 03_EDA_Hypothesis.ipynb | EDA charts and 3 hypothesis tests |
| 04_KNN_Engine.ipynb | KNN matching engine, Precision@4 evaluation |
| 05_Recommender.ipynb | 6-rule programme recommender |
| 06_Donor_Transparency.ipynb | Zakat audit and impact calculation |
| KHCC_Dataset_v4_Accurate 2.xlsx | Raw dataset (111 patients, 250 donors, 378 matches) |
| patients_clean.csv | Cleaned patient profiles |
| donors_clean.csv | Cleaned donor profiles |
| matches_clean.csv | Cleaned match history |
| KHCC_AidConnect_App_1.html | Interactive mobile app demo |
| KHCC_Dashboards_FINAL.twb | Tableau dashboard workbook |
